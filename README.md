# Morse Code

> This Method will give you back Treasure.

## Built With

- Ruby

## Getting Started

To get a local copy up and running follow these simple example steps in your terminal.

- `git clone https://github.com/Leandro-Barretoo/morse_code`
- `cd morse_code`
- `ruby bin/main.rb`
